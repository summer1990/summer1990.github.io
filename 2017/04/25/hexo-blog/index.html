使用Hexo和Github搭建blog

发表于 2017-04-25

最近想搭建一个自己的blog，记录所学到的东西。找了相关的框架和搭建方法，最终决定使用Hexo和Github搭建一个免费的blog网站。第一篇文章就记录一下我搭建博客的过程。

安装配置Hexo

Hexo是使用Nodejs开发的一个简洁、高效的博客框架，之所以选择使用Hexo，主要原因是因为我对Nodejs很感兴趣，希望借此可以对Nodejs有所了解。

废话不多说，下面就开始安装Hexo。

打开http://localhost:4000可以看到默认的blog样式和内容。

Hexo的配置文件为__config.yml。详细介绍请移步官网配置页面。
这里仅对个别配置进行设置：

属性 描述
title 网站标题
subtitle 网站副标题
description 网站描述
author 作者名字
language 网站使用的语言, 中文为zh-Hans

安装配置Next主题

由于Hexo默认的主题并不好看，所以我们采用Next主题插件。

在blog目录下

为了使Next主题能够生效，接下来需要配置两个__config.yml文件：一个是blog目录下的站点配置文件；另一个是themes/next目录下的主题配置文件。

针对blog目录下的站点配置文件__config.yml，作以下修改

属性 值
theme next

针对themes/next目录下的主题配置文件__config.yml，作以下修改

属性 值 描述
scheme Pisces 这个外观我喜欢
copyright false 不显示页面底部的copyright
use_motion false 关闭动画效果

其他个性化设置可以参照Next官网配置。
